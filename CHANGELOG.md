# Changelog

## 0.2.0
- `prefab` 工具新增 `create` 动作：从零创建 Prefab（支持根组件 + 子节点层级）
- `prefab` 工具 `modify_contents` 新增 `child_path` 参数：支持修改指定子节点（组件添加/删除、属性和 Transform 调整）
- `prefab` 工具 `create` 与 `create_from_gameobject` 统一支持 `components_to_add` + `create_child` 参数规范

## 0.1.0
- 初始化 package 脚手架与 asmdef 分层。
